1. Java 7 SDK 
-- Install latest Java 7 SE SDK form Oracle (Java 7 update 9)
-- (be aware of your operating system 32 or 64)
2. Apache Maven
-- Get the latest Apache Maven 3 (3.0.4)
-- Upzip it and place in C:\ (or anywhere you like)
-- Edit settings.xml in conf/
-- find "<localRepository>", and add our own local repository
-- "<localRepository>C:/mvn_repo</localRepository>" (optional)
3. Apache Tomcat
-- Get the latest Apache Tomcat 7 (7.0.32)
-- Upzip it and place in C:\ (or anywhere you like)
4. Eclipse
-- Get Eclipse EE (Juno)
-- Edit eclipse.ini
-- Insert
-- "-vm
-- C:/Program Files/Java/jdk1.7.0_09/bin/javaw.exe"
-- after "openFile"
5. Subclipse and m2e
-- Open Eclipse
-- Goto Help -> Install New Software...
-- Click Add... to add a repository with
-- Name: subclipse
-- Location: http://subclipse.tigris.org/update_1.8.x
-- Choose the above site, and install everything within
-- Choose All site, and find and install m2e - Maven Integration for Eclipse
6. Config m2e
-- Open Eclipse
-- Goto Window -> Preference
-- Goto Maven -> User Settings
-- Set the settings file to the file in Step 2
7. Config subclipse
-- Open Eclipse
-- Goto Window -> Preference
-- Goto Team -> Ignored Resources
-- Add three more patterns (.setting*, .project, .classpath)
-- (will not be necessary after we configure the svn file)
-- Click Apply
8. Config Tomcat
-- Open Eclipse
-- Goto Window -> Preference
-- Goto Server -> Runtime Environment
-- Click Add...
-- Choose Apache Tomcat v7.0
-- Check create new local server
-- Set the Tomcat installation directory in Step 3
-- Set our own jdk for the JRE
9. Install m2e-wtp
10. Install thyemeleaf
-- Goto Help -> Install New Software...
-- Click Add... to add a repository with
-- Name: subclipse
-- Location: http://www.thymeleaf.org/eclipse-plugin-update-site/
-- Choose the above site, and install everything within
